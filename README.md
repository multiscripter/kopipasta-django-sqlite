# Копипаста
### категории:
- Мытищи
- Страх и ненависть в Лас-Вегасе

framework: **Django**

data storage: **sqlite**

testing: **unittest**

### Run from project root:

#### Using Docker container:
- docker-compose build
- docker-compose up -d

**Run app:**
```
http://127.0.0.1:8000/
```

**dependencies**
- Python==3.8.7
- django==3.2
